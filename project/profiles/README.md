# CodeBTI Project Profiles

Project profiles summarize cross-cutting workflow preferences that apply before language-specific style profiles.

Use this profile layer after the project-wide questions are complete and before generating final guidance. Combine it with one or more language profiles, for example `Controlled Multi-Language Maintainer with Python Data-First Validator traits`.

## Inference Inputs

Use these inputs together:

- project-wide answers from [../questions/fixed-project.md](../questions/fixed-project.md),
- selected language packs and language-specific answers,
- adaptive answers from [../../shared/questions/adaptive-question-guide.md](../../shared/questions/adaptive-question-guide.md),
- contradictions or weak signals recorded in [../../shared/records/session-record.template.md](../../shared/records/session-record.template.md).
