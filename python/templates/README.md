# CodeBTI Templates

This folder contains templates for generated CodeBTI outputs.

- [CodeStyle.template.md](CodeStyle.template.md): project-specific coding style guide.
- [SKILL.template.md](SKILL.template.md): reusable agent behavior distilled from a CodeBTI result.
- [SPEC.template.md](SPEC.template.md): project-specific requirements or implementation spec.

Use a completed session record from [records/](../records/README.md) and profile inference from [profiles/](../profiles/README.md) before filling these templates.
