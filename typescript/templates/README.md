# CodeBTI Templates

This folder contains templates for generated CodeBTI outputs.

- [CodeStyle.template.md](CodeStyle.template.md): project-specific coding style guide with shared project rules before TypeScript-specific guidance.
- `SKILL.template.md`: **moved to** [../../shared/templates/SKILL.template.md](../../shared/templates/SKILL.template.md).
- `SPEC.template.md`: **moved to** [../../shared/templates/SPEC.template.md](../../shared/templates/SPEC.template.md).

Use a completed session record from [records/](../records/README.md), project profile inference from [../../project/profiles/](../../project/profiles/README.md), and TypeScript profile inference from [profiles/](../profiles/README.md) before filling these templates.
