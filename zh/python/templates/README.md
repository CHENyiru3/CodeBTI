# CodeBTI Templates

本文件夹包含生成式 CodeBTI 输出的模板。

- [CodeStyle.template.md](CodeStyle.template.md)：项目专属的编码风格指南。
- [SKILL.template.md](SKILL.template.md)：从 CodeBTI 结果提炼的可复用 Agent 行为。
- [SPEC.template.md](SPEC.template.md)：项目专属需求或实现规范。

填写这些模板前，使用 [records/](../records/README.md) 中已完成的会话记录和 [profiles/](../profiles/README.md) 中的 Profile 推断。
