# CodeBTI Templates

本文件夹包含生成式 CodeBTI 输出的模板。

- [CodeStyle.template.md](CodeStyle.template.md)：项目专属编码风格指南，先写共享项目规则，再写 Python 专属指导。
- **SKILL.template.md**：已移至 [../../shared/templates/SKILL.template.md](../../shared/templates/SKILL.template.md)。
- **SPEC.template.md**：已移至 [../../shared/templates/SPEC.template.md](../../shared/templates/SPEC.template.md)。

填写这些模板前，使用 [records/](../records/README.md) 中已完成的会话记录、[../../../project/profiles/](../../../project/profiles/README.md) 中的项目 Profile 推断，以及 [profiles/](../profiles/README.md) 中的 Python Profile 推断。
