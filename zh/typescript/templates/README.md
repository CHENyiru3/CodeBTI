# CodeBTI 模板

本文件夹包含生成的 CodeBTI 输出的模板。

- [CodeStyle.template.md](CodeStyle.template.md)：项目专属编码风格指南，先写共享项目规则，再写 TypeScript 专属指导。
- **SKILL.template.md**：已移至 [../../shared/templates/SKILL.template.md](../../shared/templates/SKILL.template.md)。
- **SPEC.template.md**：已移至 [../../shared/templates/SPEC.template.md](../../shared/templates/SPEC.template.md)。

在填写这些模板之前，使用 [records/](../records/README.md) 中的已完成会话记录、[../../../project/profiles/](../../../project/profiles/README.md) 中的项目 Profile 推断，以及 [profiles/](../profiles/README.md) 中的 TypeScript Profile 推断。
